# TT Buffer Simulation for 3D Torus Ethernet Network

Репозиторий содержит программную реализацию дискретно-событийной имитационной модели, разработанной в рамках выпускной квалификационной работы:

**«Методы управления трафиком реального времени в центре обработки данных с трёхмерной тороидальной топологией»**

Модель используется для проверки аналитических оценок размера TT-буфера и задержки доставки приоритетного TT-трафика в Ethernet-сети ЦОД с топологией 3D-тор.

## Краткое описание модели

В модели реализованы:

- топология сети 3D-тор;
- статическая XYZ-маршрутизация;
- генерация синтетических TT-потоков и фонового трафика;
- ограниченные TT-буферы на выходных портах узлов коммутации;
- сценарии обслуживания TT-трафика:
  - SP+FP;
  - TAS;
  - CQF;
- сбор основных метрик моделирования:
  - число сгенерированных TT-кадров;
  - число сброшенных TT-кадров;
  - наблюдаемая доля потерь;
  - средняя сквозная задержка;
  - максимальная сквозная задержка.

## Требования
Для запуска модели требуется Python 3.10 или выше.

Внешние зависимости не требуются: программа использует только стандартные библиотеки Python.

## Запуск модели
Склонировать репозиторий

- git clone https://github.com/george-vak/tt-buffer-3d-torus.git

- cd tt-buffer-3d-torus

Запустить моделирование 

- python3 mmm.py
- python mmm.py, если используется Windows
